## Audit détaillé

### 1. Exactitude factuelle (9/10)

**Points forts** :
- Toutes les données chiffrées clés sont sourcées et recoupées (déficit 2 025 MW, PIB -4%, taux 510 CUP/USD, 2,7M départs)
- L'Executive Order 14380 est correctement référencé avec sa date et son cadre juridique (IEEPA + NEA)
- Les citations sont attribuées (Guterres, Trump, Rubio)
- La distinction entre chiffres officiels et estimations indépendantes est clairement établie (population 8,0-8,6M vs 10,5M officiel)

**Points d'attention** :
- L'inflation "réelle" de 30-40% est une estimation analytique, pas un chiffre sourcé — correctement signalé comme "probablement" mais pourrait être plus explicite
- Le chiffre de 20 000 Cubains dans l'armée russe provient des renseignements ukrainiens via Heritage Foundation — source potentiellement biaisée, mentionné mais non qualifié dans l'article

**Recommandation** : Aucune correction nécessaire, les nuances sont correctement signalées.

### 2. Qualité du sourcing (8,5/10)

**Points forts** :
- 48+ sources consultées, ratio A/B excellent (41 sur 48)
- Diversité géographique (US, UE, Chine, Russie, Amérique latine)
- Sources institutionnelles de premier rang (White House, ONU, HRW)
- Think tanks variés (Foreign Affairs, Cato, Heritage, IFRI)
- Sources juridiques spécialisées (Greenberg Traurig, Baker McKenzie)

**Points d'attention** :
- CiberCuba (fiabilité C) est la source principale pour les données temps réel (taux de change, blackouts) — peu d'alternatives disponibles
- Absence de sources du FMI et de la Banque mondiale (Cuba non-membre du FMI — lacune structurelle correctement documentée)
- Pas de source primaire cubaine officielle (Granma, ONEI) — biais potentiel, mais accès limité

**Recommandation** : Score conforme. La lacune sur les sources cubaines officielles est inhérente au sujet.

### 3. Structure & lisibilité (9/10)

**Points forts** :
- Structure Deep Dive respectée : synthèse exécutive → thèse → diagnostic → dynamiques → scénarios → recommandations
- Hiérarchie claire avec parties numérotées (I à V)
- Key figures en haut de page pour lecture rapide
- Insight boxes bien utilisés (alert pour risques, gold pour signaux stratégiques)
- Tableau SEMPLICE intégré dans l'article
- Matrice sectorielle pour les PME/ETI

**Points d'attention** :
- Article long (~3 500 mots) — dans la fourchette haute du format Deep Dive (2 000-4 000 mots)
- La section V (recommandations) pourrait être plus concise

**Recommandation** : Aucune modification nécessaire. La longueur est justifiée par la complexité du sujet.

### 4. Pertinence décisionnelle (9/10)

**Points forts** :
- Trois scénarios avec probabilités chiffrées (10-15%, 50-55%, 30-35%)
- Matrice d'impact sectoriel concrète (7 secteurs, niveaux de risque, recommandations)
- Recommandations opérationnelles structurées par horizon temporel (immédiat/CT/MT)
- Indicateurs d'alerte avec seuils critiques mesurables
- Mention explicite d'Air France, Accor, Pernod Ricard, TotalEnergies, Schneider Electric — entreprises françaises concernées

**Points d'attention** :
- Les recommandations sont principalement défensives (gel, évacuation, diversification) — peu de pistes offensives pour les acteurs ayant une appétence au risque
- Le scénario de reconstruction post-transition mériterait plus de développement

**Recommandation** : Conforme à la situation de risque critique. Les pistes offensives sont prudemment mentionnées en MT.

### 5. Neutralité analytique (8/10)

**Points forts** :
- Ton analyste stratégique respecté, pas journaliste sensationnaliste
- Les incertitudes sont signalées ("probablement", "estimation", "les données disponibles suggèrent")
- Les contradictions OSINT sont traitées (cause de la crise : endogène + exogène)
- Pas de jugement moral sur le régime cubain ni sur la politique américaine

**Points d'attention** :
- Le terme « siège américain » dans le titre est analytiquement correct mais connoté — pourrait être perçu comme non-neutre par certains lecteurs
- La rhétorique Trump est abondamment citée ; la position cubaine officielle est moins représentée (contrainte de sourcing)
- Le « friendly takeover » est présenté sans guillemets dans le titre — corrigé dans le corps de l'article

**Recommandation** : Score acceptable. Le sous-titre et le corps de l'article compensent la tonalité du titre principal.

- Navigation, Footer, MarketTicker identiques

**Recommandation** : Pleinement conforme.
